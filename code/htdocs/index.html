Instructions

Measuring. Just hit the measurement button on the home screen and new data will appear in the sensor sections.

Data Conversion. This is a prototype, and displayed weights are not properly converted to kilograms yet, as the strain gauges were not available for calibration.

Colour-Coding. The sensor fields are colour-coded as follows to indicate the status of each sensor.
Green: Data has successfully been received.
White: A measurement has been requested and no response has been received from the server yet. If this state persists for all sensors, try to send a new measurement request.
Red: The sensor has not responded to the most recent request. The previous data value is displayed. Try resending the request, and check that all sensor units are turned on.
